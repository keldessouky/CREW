# Repo Skill Readiness

This repository contains a repeatable process for evaluating repository support readiness, identifying required skills, and building redundancy across engineering teams.

## Included artifacts
- `SKILL.md` — workflow for generating repeatable repo skill readiness reports
- `template.md` — reusable report template for repo readiness assessments
- `atom-repo-skill-readiness-summary.md` — sample executive summary for the Atom repository

## Usage
1. Review `template.md` for the evaluation framework.
2. Use `SKILL.md` as the workflow definition for the report.
3. Apply the same process to additional repos owned by the team.
